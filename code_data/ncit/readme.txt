These are the steps to make it run:
1. Run NCIT_Data.sql script in your local sql server
2. Update <connectionStrings> in CTEligible.exe.config file in the binaries folder as below
   a)<ServerName> needs to  be replaced with your server name
   b)<password> needs to be replaced with your sql password
   
3. Run CTEligible.exe
4. Browse to the file location 'cteligible.txt' (download from github) in your local drive and upload it.
